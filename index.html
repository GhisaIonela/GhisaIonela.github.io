PolyScape Creator

PolyScape Creator is a 3D world-building game developed using the Godot game engine. It allows users to create custom low-poly environments using a tile-based system.

  • Within the tile building screen, players can adjust various properties of the tiles, including dynamic rotation for added realism. This feature enables the creation of complex structures such as stairs and slopes that seamlessly integrate into the landscape.
  • The game features a day phase slider to manipulate the atmosphere, transitioning from day to night with changing sky colors. This adds depth and immersion to the virtual environments.
  • Navigation controls are intuitive, allowing players to move the camera using the WASD keys, zoom in and out, and rotate the view with the middle mouse button.
  • Tile placement is facilitated by ray-casting, ensuring precise positioning during construction. When users left-click and drag the mouse, a selection is formed for the upcoming placement of tiles once the left click is released. Conversely, right-clicking and dragging the mouse will reveal an advanced options menu for the created selection, providing players with choices to delete or insert tiles.

HyPhal

HyPhal is a mini social networking application developed by a team of two using IntelliJ IDEA with Java for the backend and frontend and PostgreSQL for the database. The app follows a client-server architecture, with the backend initially developed and packaged into a JAR file before integration with the frontend.

  • Key features of HyPhal include user account management, password encryption for secure storage in the database, and password recovery functionality. Users can log in to their accounts, access their profile pages, and navigate to various sections such as the friends page, direct messages, and inbox.
  • On the friends page, users can browse profiles, send friend requests, accept or deny pending requests, cancel sent requests, and unfriend existing connections. The app maintains a history of friend requests and updates users in real-time on pending requests and incoming messages.
  • Direct messaging functionality allows users to communicate with friends both within and outside their friend list. Users can send messages, reply to messages, and engage in ongoing conversations.
  • HyPhal is designed to handle concurrent instances, ensuring smooth operation and real-time updates across multiple sessions.

Cookpedia

Cookpedia is a recipe management platform developed using HTML, CSS, and JavaScript for the client-side functionality. PHP was utilized for server-side scripting, while MySQL served as the database management system. The site is hosted using XAMPP, providing a reliable hosting solution for the platform.

  • Cookpedia is designed to cater to guests, logged users, and administrators. Guests have access to the community tab, where they can view recipes posted by the Cookpedia community.
  • Logged users gain additional functionality, including the ability to create and manage their own cookbooks in the cookbook tab. Cookbooks allow users to add recipes, providing details such as recipe name, description, ingredients, instructions, and images of the finished dishes.
  • Users can also manage their account details in the account tab, where they can view account information and select from six available themes for the app's appearance.
  • Administrators have access to the admin dashboard tab, where they can moderate user-submitted recipes by editing or deleting them as necessary.
